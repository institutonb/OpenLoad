Aprenda Construir Apps onde irei levá-lo passo-a-passo através do processo de construção de aplicativos diferentes em iOS 7 e XCode 5.

Nossa formaçaõ ensina as ferramentas necessárias para desenvolver aplicações para o iPhone e iPad para iOS7.

Ao longo de nossa jornada, vamos aprender a sintaxe do Objective-C, a linguagem utilizada para desenvolver para iOS, bem como padrões de projeto importantes e melhores práticas.

Ao final do curso, você deve ser capaz de entender e recriar muitas das características vistas em aplicações iOS populares e estender esse conhecimento para fazer aplicativos de sua preferência.

Acreditamos que os alunos aprendem através da construção.

É por isso que temos de construir aplicativos durante o curso, com o aplicativo final que é um aplicativo completo.


Temas Abordados:
  • * Objective-C: Aprenda a linguagem usada para desenvolver todas as iOS nativas e aplicativos Mac
  • * Analisa:Uma nova ferramenta revolucionária que permite aos desenvolvedores iOS para criar um servidor em Objective-C
  • * iOS7 e XCode 5: Saiba as últimas plataformas tecnológicas da Apple
  • * API Integração: Conecte-se com APIs de terceiros, como o Facebook, Foursquare e mais
  • * Storyboard: Saiba como criar interfaces bonitas usando construtor de interface gráfica da Apple
  • * Core Data: Armazenar dados em seu dispositivo com o banco de dados central
  • * Git: Aprenda a nós Git para controle de versão e colaboração
  • * Mixpanel: Integrar Analytics em seus aplicativos
Por que fazer este curso e quanto tempo leva para completar?

Desenvolvimento iOS, e software como um todo, é uma carreira em franca expansão.

A demanda por novos desenvolvedores de todos os níveis supera de longe a oferta.

Barreira à entrada não é tão elevado como se poderia pensar - embora a paixão e o trabalho duro são necessários, uma licenciatura em ciência da computação ou qualquer outro domínio técnico não é necessário.

Nosso objetivo é levá-lo a partir de iniciante para o desenvolvedor júnior em 4 meses.

Os aplicativos criados neste curso pode ser usado em sua carteira, então você tem algo para mostrar quando você se candidatar a empregos.

Iniciantes (sem experiência de codificação necessária).
